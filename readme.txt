== Installation ==

= Minimum Requirements =

- PHP 7.4
- WordPress 6.0

= Automatic Installation =

1. Log in to your WordPress admin dashboard.
1. Navigate to the "Plugins" menu.
1. Click on "Add New".
1. Search for "Tracktastic".
1. Click on "Install Now" and then on "Activate".

= Manual Installation =

1. Download the plugin.
1. Upload it to your web server using an FTP application.

= Once Installed =

The plugin has no configuration screen, as no configuration is required. If Matomo tracking is set correctly and WooCommerce is activated, your sales statistics will appear in Matomo.

== Frequently Asked Questions ==

= Is this the official plugin proposed by the Matomo team? =

No. This plugin is not developed by the team behind Matomo. It was developed by a user who needed a solution for tracking WooCommerce statistics with Matomo and decided to share his plugin with the WordPress community.

= Does Tracktastic allow the installation of Matomo? =

No, Tracktastic does not facilitate the installation of Matomo. It exclusively enables the e-commerce tracking functionality within your WooCommerce store for integration with an existing Matomo setup. We recommend setting up Matomo on your WordPress site with the [SEOPress extension](https://fr.wordpress.org/plugins/wp-seopress/ "Optimize your WordPress SEO").

= How do I activate Matomo's e-commerce tracking with Tracktastic? =

Activating e-commerce tracking is straightforward. Simply install and activate the Tracktastic extension on your WordPress site. Ensure you have the basic Matomo script already in place on your site, and Tracktastic will handle the rest.

= Can I use Tracktastic without WooCommerce? =

No, Tracktastic is specifically designed to collect and track e-commerce events from WooCommerce. Without WooCommerce, Tracktastic will not function as it relies on WooCommerce data for its operations.

= Does Tracktastic work with Matomo On-Premise? =

Yes, Tracktastic is compatible with both Matomo On-Premise and Matomo Cloud. It seamlessly integrates with your Matomo setup, regardless of whether it's hosted on-premise or in the cloud.
